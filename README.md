# RITZ Media Assignment

Frontend assignment built for the **RITZ Media** hiring process using **Next.js App Router**, **TypeScript**, and **Tailwind CSS**. The goal was to recreate the provided Figma homepage with a strong focus on responsive design, smooth animations, and clean component architecture.

## 🔗 Live Demo

- **Live:** https://ritz-media-assignment.vercel.app/
- **GitHub:** https://github.com/Nitishojha00/RITZ-MEDIA-Assignment

## 🚀 Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- GSAP
- Framer Motion
- Lenis (Smooth Scrolling)

## ✨ Features

- Pixel-perfect implementation based on the Figma design
- Smooth scrolling with premium animations
- Scroll-triggered reveal animations
- Hover & micro-interactions
- Fully responsive (Desktop, Tablet & Mobile)
- Reusable component-based architecture
- Optimized performance and clean code

## 📂 Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/Nitishojha00/RITZ-MEDIA-Assignment.git

cd RITZ-MEDIA-Assignment
npm install
npm run dev
```

Open **http://localhost:3000** in your browser.

## 📌 Assignment Objective

- Recreate the provided Figma homepage
- Deliver smooth, premium-quality animations inspired by the reference website
- Ensure responsiveness across all screen sizes
- Follow modern React and Next.js best practices
